命名补充

  • 文件名必须以大写字母打头
  • class名，必须按文件路径
    e.g. app/view/test/Test.js == appName.view.test.Test
  • view 文件夹下 xtype 以小写字母打头，且带有小尾巴 -View
    e.g. xtype:’testView’
  • alternateClassName 也必须以大写字母开头
  • 日期时间使用格式 “201703241148Z” 2017年3月24日11点48分 尾巴Z是跟CG API
