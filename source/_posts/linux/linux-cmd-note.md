```
$ls -l

$less file

/ 向下查找

```

正则表达式搜索文件:
```
$egrep -i -a1 'loadWithdrawalacc[a-zA-Z -]+0324'  2goportal.log
 -i 忽略大小写
 -b1 匹配前一行
 -a1 包含被匹配后一行

```
